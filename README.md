# AngularMusic

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.3.

## Requirements

- [x] Songs
- [x] Lists
- [x] Add List
- [x] Add Song to List
- [x] Delete List
- [x] Delete Song from List
- [x] Show more Songs
- [ ] Order By/Sort Songs (Custom pipe needed)
